webpack

2020 7-15

1. 概念

本质上，webpack是一个javascript应用程序的静态模块打包器。
当webpack处理应用程序时，递归地构建一个依赖关系图，其中包含应用程序的每个模块，打包成一个或多个bundle。

2.入口(entry)

构建依赖的开始。进入入口起点后，webpack会找出哪些模块和库的直接或间接的依赖。
每个依赖项随即被处理，输出到bundle文件中。

//单个入口
module.exports = {
    entry:{
        main:"src/main.js"
    }
}
//单个入口简写
module.exports = {
    entry:"src/main.js"
}
//多个入口
module.exports = {
    entry:{
        entryChunkName1:"src/a.js"
        entryChunkNam2:"src/b.js"
    }
}

3. 出口（output）

module.exports = {
    output:{
        filename:'bundle.js', //输出文件名
        path:'/dist', //输出的绝对路径
    }
}
//配置多入口起点，多出口
module.exports = {
    entry:{
        a:'src/a.js',
        b:'src/b.js'
    },
    output:{
        filename:'[name].js',
        path:'/dist'
    }
} 
//生成文件 dist/a.js dist/b.js

4. mode

module.exports = {
    mode:"development" //production 二选一
}
