搭建简单主题

2020-07-14

1. 创建目录结构

themes
 ┃
 ┖───themes name
        ┇
        ┠┄┄┄layout
        ┇       ┖─index.ejs //主页
        ┇       ┖─aside.ejs //侧边栏
        ┇       ┖─layout.ejs //框架
        ┇       ┖─nav.ejs   //导航
        ┇       ┖─post.ejs  //post文章展示格式
        ┠─┄source //资源
        ┇        ┖─css //css 文件
        ┇        ┖─js //js文件
        ┇        ┖─images //img
        ┇        ┖─lib //第三包  
        ┖──_config.yml //主题配置文件

2.改主题

#项目_config.yml

theme: themes name

3.基础ejs语法

  • a. 主要使用标签

    • <% 脚本标签，用于流程控制，无输出
    • <%__ 删除其前面的空格符
    • <%= html转义输出到模板
    • <%- 输出非转义数据到模板
    • <%# 注释
    • <%% 输出字符串<%
    • %> 一般结束标签
    • _%> 删除紧随其后的空格
    • __%> 删除其后的空格

  • b. includes

    通过include指令将相对于模板路径中的模板片段包含进来。
    <%- includes('a/b') %>

4. hexo 变量

变量 描述
site 网站变量
page 针对该页面的内容以及front-master所设定的变量
config 网站变量
theme 主题配置
_ Lodash库
path 当前页面的路径（不含根路径）
url 当前页面的完整路径
env 环境变量

5. 资源文件夹

source文件夹中除了文章以外的所有资源，如图片、css、js等，
如图a.jpg在images中使用 (/images/a.jpg)

6. 字体图标

图标库下载图标，放入source/lib。

<%- css('/lib/iconfont/iconfont.css') %>

7. 引入highlight

下载highlight

<!-- 引入css -->
<head>
    <%- css('/lib/highlight/styles/atom-one-dark') %>
</head>
<!-- 引入js -->
<%- js('/lib/highlight/highlight.pack.js') %>
<!-- 使用 -->
<script>hljs.initHighlightingOnLoad();</script>
