Hexo + Gitbub搭建

2020-07-08

1. 下载hexo

$ npm i hexo-cli -g

2. 初始化项目

$ hexo init <folder>
$ cd <folder>
$ npm install

3. github创建 repositories

命名 name.github.io
name 必须与github仓库名相同

4. 绑定github地址

//_config.ym
deploy:
  type: git
  repository: git地址（https/ssh都行）
  branch: master

5. 生成静态文件

$ hexo g

6. 上传到github

$ hexo d

7. 访问地址

https://name.github.io

8. 清除上次缓存

$ hexo clean

9. 启动本地服务

$ hexo s
